## 🚀 Para usar o sistema:

### 1. Instalar dependências:
```bash
npm install
```

### 2. Configurar MySQL:
- Instale o MySQL se não tiver
- Crie um banco chamado `ddrinks_db`
- Edite `config.env` com suas credenciais

### 3. Inicializar sistema:
```bash
npm run setup
```

### 4. Executar:
```bash
npm run dev
```

### 5. Acessar:
- Abra: http://localhost:3000
- Cadastre-se e comece a usar!
